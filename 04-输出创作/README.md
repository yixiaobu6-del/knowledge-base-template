# 04-输出创作

## 用途

所有对外产出内容的文件夹，从灵感到成品。

## 目录说明

| 子目录 | 用途 | 示例 |
|--------|------|------|
| `文章/` | 已完成的成品文章 | 如何搭建个人知识体系 |
| `草稿/` | 进行中的创作 | 关于AI的思考（草稿） |
| `作品集/` | 所有作品的汇总索引 | 2024文章列表 |
| `灵感碎片/` | 随手记录的创意和想法 | 一个有意思的idea |

## 使用规则

- **文章**：只放已发布的完整内容，每篇一个文件
- **草稿**：进行中的文章，文件名加 `_草稿` 后缀
- **作品集**：按年份或主题建立索引文件
- **灵感碎片**：一句话一个文件，标注日期和关联领域

## 创作流程

```
灵感（04-灵感碎片）
  → 大纲（04-草稿）
  → 初稿（04-草稿）
  → 修改（04-草稿）
  → 发布（04-文章/）
  → 归档（05-复盘归档）
```

## 文件命名

```
YYYY-MM-DD_标题.md          # 单篇文章
YYYY-MM_系列名称_序号.md     # 系列文章
作品集_2024.md               # 年度作品索引
```
