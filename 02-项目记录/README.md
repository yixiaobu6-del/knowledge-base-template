# 02-项目记录

## 用途

跟踪所有项目的进展状态，从启动到完结的全生命周期记录。

## 目录说明

| 子目录 | 用途 | 示例 |
|--------|------|------|
| `进行中/` | 正在进行的项目文档 | 个人网站 V2 搭建 |
| `已完成/` | 已完结的项目归档 | 2024年度报告 |
| `待办/` | 项目级任务清单 | 书单清单 |
| `会议记录/` | 项目相关会议纪要 | 1月项目同步会 |

## 使用规则

- 每个项目一个独立 Markdown 文件
- 项目状态变化后及时移动到对应目录
- 已完成的项目保留关键决策记录
- 会议记录要有行动项和负责人

## 项目文件模板

```markdown
# 项目名称

**状态**：进行中 / 已完成 / 暂停
**起止时间**：YYYY-MM-DD ~ YYYY-MM-DD
**标签**：#type/项目记录 #领域/XXX

## 目标
...

## 里程碑
- [ ] M1: ... (截止日期)
- [ ] M2: ...

## 关键决策记录
| 日期 | 决策内容 | 原因 | 影响 |
|------|----------|------|------|

## 复盘总结
...
```
