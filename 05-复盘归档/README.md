# 05-复盘归档

## 用途

定期复盘，让经验沉淀为知识。同时对过期笔记进行归档管理。

## 目录说明

| 子目录 | 用途 | 内容 |
|--------|------|------|
| `周复盘/` | 每周回顾与反思 | 做了什么、学到了什么、下周计划 |
| `月复盘/` | 月度总结与方向调整 | 基于周复盘的综合梳理 |
| `年复盘/` | 年度回顾与来年规划 | 全面总结+下年OKR |
| `归档/` | 过期的旧笔记 | 不删除但移到此处保管 |

## 复盘模板结构

### 周复盘

```markdown
# YYYY-WW 周复盘

## 本周完成
- 工作：[完成内容]
- 学习：[完成内容]
- 生活：[完成内容]

## 亮点与问题
- 亮点：...
- 问题：...

## 关键思考
...

## 下周计划
1. ...
```

### 月复盘

```markdown
# YYYY-MM 月复盘

## 数据统计
- 完成项目数：
- 阅读量：
- 输出文章数：

## 目标回顾
- 本月目标完成度：
- 未完成原因：

## 经验沉淀
- 什么做得好：...
- 什么可以改进：...
```

## 归档规则

- 超过1年无更新的笔记 → 移入归档
- 已完结项目的原始记录 → 移入归档
- 归档只移不删，保留完整历史
