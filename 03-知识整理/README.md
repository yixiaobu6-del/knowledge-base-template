# 03-知识整理

## 用途

核心知识库，每个笔记都是一个独立的知识单元。通过 [[双向链接]] 连接形成知识网络。

## 目录说明

| 子目录 | 用途 | 示例 |
|--------|------|------|
| `概念笔记/` | 一个概念的完整理解 | 复利效应、网络效应 |
| `思维模型/` | 可复用的思维框架 | 第一性原理、奥卡姆剃刀 |
| `方法论/` | 可执行的流程方法 | 费曼学习法、GTD |
| `专题研究/` | 深度研究某个主题 | AI对教育的影响 |
| `人物档案/` | 值得学习的人的档案 | 查理芒格、瑞达利欧 |
| `常青笔记/` | 持续迭代更新的笔记 | 我的创作原则 |

## 笔记规范

- **一篇笔记只讲一个概念**
- 用 `[[双向链接]]` 连接相关笔记
- 用自己的话重述，不要搬运
- 标注信息来源
- 持续迭代更新，每次更新标注日期

## 标签体系

```markdown
#type/概念笔记      笔记类型
#type/思维模型
#type/方法论
#type/专题研究
#type/人物档案
#type/常青笔记

#status/初稿         笔记状态
#status/待更新
#status/已完成
```
