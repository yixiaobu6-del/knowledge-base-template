# assets

## 用途

存放图片、PDF、音频等附件资源。

## 目录结构建议

```
assets/
├── images/        # 图片文件
├── pdfs/          # PDF文档
├── audio/         # 音频文件
└── other/         # 其他附件
```

## 命名规范

```
YYYY-MM-DD_描述_序号.扩展名
```

示例：
- `2025-06-01_架构图_v2.png`
- `2025-06-15_会议纪要.pdf`

## 使用说明

- 附件尽量压缩后保存
- 超过10MB的文件考虑外部存储
- 在笔记中通过 `[描述](assets/images/xxx.png)` 引用
- 删除笔记时同步清理无用附件
