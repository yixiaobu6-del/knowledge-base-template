# 个人知识库架构模板

知识库结构乱七八糟？
**模板帮你设计清晰架构。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()
[![Obsidian](https://img.shields.io/badge/Obsidian-兼容-purple)]()

## 知识库不再"乱"

- **架构设计**：知识库结构可视化设计
- **Obsidian导出**：JSON+Shell脚本一键生成
- **健康诊断**：空文件夹/缺失模板检测
- **模板生成**：自动生成知识库模板文件

## 怎么用

打开 `kb_designer.html` → 设计结构 → 导出Obsidian配置
