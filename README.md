# 个人知识库架构模板

一套完整的个人知识管理体系，涵盖信息输入、项目记录、知识整理、输出创作、复盘归档的全流程。

## 核心理念

### 闭环流转

```
外部信息 → 01-信息输入 → 03-知识整理 → 04-输出创作 → 05-复盘归档
                ↓              ↑
            02-项目记录 ───────┘
```

### 双链驱动

所有笔记通过 [[双向链接]] 和标签系统连接，形成知识网络而非孤立的文件堆。

## 目录详细说明

### 01-信息输入

**用途：** 收集从外部获取的有价值信息，这里只做**记录**不做**整理**。

| 子目录 | 用途 | 示例 |
|--------|------|------|
| 阅读笔记 | 读完书后的笔记 | 《思考快与慢》笔记 |
| 播客笔记 | 听完播客的要点 | 知行小酒馆 EP23 笔记 |
| 课程笔记 | 学习的课程 | Coursera ML课程 |
| 社交媒体 | 刷到的精华内容 | 即刻、Twitter收藏 |
| 收件箱 | 最原始的碎片信息 | 一张截图、一段录音 |

**使用规则：**
- 收件箱每天清理一次
- 其他目录每周整理一次
- 只记录、不批判

### 02-项目记录

**用途：** 跟踪所有项目的进展状态。

| 子目录 | 用途 | 示例 |
|--------|------|------|
| 进行中 | 正在做的项目 | 个人网站搭建 V2 |
| 已完成 | 做完了的项目 | 2024年度报告 |
| 待办 | 任务列表 | 书单清单 |
| 会议记录 | 会议纪要 | 1月项目同步会 |

### 03-知识整理

**用途：** 核心知识库，每个笔记都是一个独立的知识单元。

| 子目录 | 用途 | 示例 |
|--------|------|------|
| 概念笔记 | 一个概念的完整理解 | 复利效应 |
| 思维模型 | 思维框架 | 第一性原理 |
| 方法论 | 可执行的方法 | 费曼学习法 |
| 专题研究 | 深度研究成果 | AI对教育的影响 |
| 人物档案 | 值得学习的人 | 查理·芒格 |
| 常青笔记 | 持续更新的笔记 | 我的创作原则 |

**使用规则：**
- 每篇笔记只讲一个概念
- 用 [[双向链接]] 连接相关笔记
- 不断迭代更新

### 04-输出创作

**用途：** 所有产出内容，从灵感到成品。

| 子目录 | 用途 | 示例 |
|--------|------|------|
| 文章 | 已完成 | 如何搭建个人知识体系 |
| 草稿 | 进行中 | 关于AI的思考（草稿） |
| 作品集 | 汇总 | 2024文章列表 |
| 灵感碎片 | 随手的灵感 | 一个有意思的idea |

### 05-复盘归档

**用途：** 定期复盘，让经验沉淀为知识。

| 子目录 | 用途 |
|--------|------|
| 周复盘 | 本周做了什么、学到了什么 |
| 月复盘 | 月度回顾、调整方向 |
| 年复盘 | 年度总结、下年规划 |
| 归档 | 过期的笔记，不删除但移到此处 |

### 06-模板脚本

**用途：** 存放笔记模板和自动化工具。

### assets

**用途：** 图片、PDF等附件资源。

## 标签系统设计

### 通用标签

```
#type/概念笔记      # 笔记类型
#type/项目记录
#type/阅读笔记

#status/进行中       # 笔记状态
#status/已完成
#status/需更新

#importance/高      # 重要程度
#importance/中
#importance/低
```

### 领域标签

```
#领域/编程
#领域/投资
#领域/写作
#领域/心理学
#领域/产品
```

### 来源标签

```
#来源/书籍
#来源/播客
#来源/课程
#来源/文章
#来源/对话
```

## 命名规范

### 文件命名

```
YYYY-MM-DD_标题.md         # 普通笔记
YYYY-MM-DD_项目名_标题.md   # 项目笔记
概念_概念名.md               # 概念笔记
人物_人物名.md               # 人物档案
```

### 标题规范

```
# 标题 - 副标题（可选）
```
