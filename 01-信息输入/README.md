# 01-信息输入

## 用途

收集从外部获取的有价值信息。**这里只做记录，不做整理。**

## 目录说明

| 子目录 | 用途 | 示例 |
|--------|------|------|
| `阅读笔记/` | 读完书/文章后的记录 | 《思考快与慢》笔记 |
| `播客笔记/` | 听完播客的要点 | 知行小酒馆 EP23 笔记 |
| `课程笔记/` | 学习的课程记录 | Coursera ML课程 |
| `社交媒体/` | 刷到的精华内容 | 即刻、微博收藏 |
| `收件箱/` | 原始的碎片信息 | 截图、录音、随手记 |

## 使用规则

- **收件箱**：每天清理一次，移到对应目录或删除
- **阅读笔记**：读完即记，不限格式
- **播客笔记**：关键观点+个人启发
- **社交媒体**：只存对自己有启发的
- **课程笔记**：按课程名建子目录

## 基本原则

> 只记录，不批判。先存进来，再决定价值。

## 归档规范

每条笔记至少包含 3 个标签，如 `#来源/书籍 #领域/产品 #status/待整理`。
