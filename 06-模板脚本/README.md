# 06-模板脚本

## 用途

存放笔记模板和自动化工具脚本，提高知识库日常维护效率。

## 目录说明

| 子目录 | 用途 |
|--------|------|
| `模板/` | 各类笔记模板（Obsidian Templater） |

## 模板清单

| 模板 | 用途 | 适用场景 |
|------|------|----------|
| 阅读笔记模板 | 记录读书要点 | 读完一本书后 |
| 会议记录模板 | 会议纪要 | 开会时 |
| 项目管理模板 | 跟踪项目 | 开始新项目时 |
| 周复盘模板 | 每周回顾 | 每周末 |
| 日记模板 | 每日记录 | 每天 |

## 自动化脚本

配合 [Obsidian自动化脚本库](https://github.com/yourname/Obsidian自动化脚本库) 使用：

```bash
# 自动整理未归类笔记
python organize.py --source ./01-信息输入/收件箱/

# 自动更新索引
python update_index.py

# 自动备份
python backup.py --destination /path/to/backup/
```

## 维护建议

模板和脚本应定期更新优化：
- 每季度审查一次模板使用率
- 废弃低使用率的模板
- 根据使用反馈迭代改进
